# @buildcanada/charts

A configurable data visualization library for creating interactive charts.

## Features

- **Multiple Chart Types**: Line charts, bar charts, stacked area, slope charts, scatter plots, and maps
- **Interactive**: Tooltips, zoom, entity selection, and time sliders
- **Configurable**: Custom branding, colors, and data sources
- **Responsive**: Adapts to container size
- **Accessible**: Keyboard navigation and screen reader support

## Quick Start

```bash
npm install @buildcanada/charts
# or
bun add @buildcanada/charts
```

```tsx
import { ChartsProvider, Grapher, GrapherState } from "@buildcanada/charts"
import "@buildcanada/charts/styles.css"

const config = {
  branding: { poweredByText: "My Organization" },
  dataApi: { baseUrl: "https://api.example.com/v1/indicators/" }
}

function App() {
  return (
    <ChartsProvider config={config}>
      <Grapher grapherState={grapherState} />
    </ChartsProvider>
  )
}
```
